# Using Node

* Interact with Node console
* Run a file with node

### Node console
node
* run node console
* execute JS code
* ctrl + c twice to quit
* no dom, no event listeners, etc

#### REPL
- Read Evaluate Print Loop

### Run a file with node
* node <filename>
* will execute file as js


#### Ruby's REPL
- irb, write ruby code
- ruby filename, to run a .rb file
